About Model Display

The team must bring a layout of the project mentioning the project name, the aim of the project, working principles, and practical applications. Also, the project's expenditure is to be mentioned on a chart paper.


Rules:

1. Dimension of the project base must not exceed (3x4) ft.

2. The team must bring a layout of the project mentioning the project name, the aim of the project, working principles, and practical applications. Also, the project's expenditure is to be mentioned on a chart paper.

3. At least two (2) group members should always remain present for the project demonstration during the exhibition period, i.e. 10.00 am - 5.00 pm.

4. Virtual projects like web applications will also be allowed for the competition.

5. Each group must carry an extension cord for Electric Connection. Otherwise, the organizing committee will be unable to provide an electrical connection to the models/ projects if required.


Team Details:

Participants should only form a group of four (4) students.


Eligibility:

All Participants should have proper College ID card on them, and should be able to produce the same in original when asked for verification, failing which they will not be allowed to participate. Each participant should be a student pursuing some field of study in college/university.


Registration Fees:

The one-time registration fee for each candidate is Rs.100/- .
