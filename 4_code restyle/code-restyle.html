About Code Restyle

CODE RESTYLE was first introduced in TECHETC 2k19. It will be an awesome platform to hone your knowledge of the internal working of the C language over 2 rounds of exciting questions. So, have you mastered the C language? Then, this is the event for you. Come join us this TECHETC 2k23 and show us your programming knowledge.

Event Details:

CODE RESTYLE will be spread over 2 rounds, the first being an MCQ round and the second a blind coding round. The details of the rounds are as follows:
⦁ Round 1 (MCQ): There will be 20 MCQ questions from C language and 10 MCQ questions from general computer aptitude.
Time-30 minutes.
⦁ Round 2 (Debugging): You will be given a solution(code) with some output sets. On the basis of these output sets, debug the code.
Time-30 minutes.


Rules:

Rules for Round 1:
⦁ The MCQ round will be on Google Form.
⦁ There will be no negative marking on wrong response.
⦁ In case of tie in scores, submission time will be considered.
⦁ Top 25% of participants (may vary) will be moving on to the next round. Rules for Round 2:
⦁ A solution (code in C language) will be given to you with some output sets.
⦁ Then, you have to debug the entire solution (in the C language) to remove all the errors (logical, syntactical etc.) from the code on the basis of the output sets so that the code can give those outputs successfully.
⦁ Scoring will be based on the number of errors corrected count.
⦁ In case of tie in scores, submission time will be considered.

The decision of the coordinators will be final.


Team Details:

CODE RESTYLE is an individual event.


Eligibility:

All participants should have proper College ID card on them, and should be able to produce the same in original when asked for verification, failing which they will not be allowed to participate. Each participant should be a student pursuing some field of study in college/university.
